Grid最新版本号1.2.1
====
## 按住鼠标右键，移动鼠标或滚动滚轮，打开功能菜单，松开右键执行操作

## 1.亮点

#### 选中文字后按住右键，向上移动鼠标进行复制，向下粘贴，非常流畅
#### 选中文字后按住右键，向上滚动鼠标滚轮进行复制，向下滚动粘贴，非常流畅
#### 按住右键向左切换到上一个窗口，向右切换到下一个窗口
#### 隐藏到托盘的功能也非常好用
#### 退出在第一格设置里
